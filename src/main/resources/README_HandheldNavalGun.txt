

	HandheldNavalGun 0.0.4


・このmodについて

	携帯型の火砲 Hand-Held Naval Gun と、その弾薬 AAC Round を追加します。


・前提環境（必須）

	Java 8
	Minecraft 1.10.2
	Minecraft Forge 1.10.2-12.18.3.2185-


・インストール方法

	JARファイルをmodsフォルダ内に置く


・追加されるアイテム

	EN: Hand-Held Naval Gun
	JP: 単装高角砲（携帯型）

		このmodのメインである武器アイテム。

		LMB: Lock-on a target.
		RMB: Shoot and Reload.

		・レシピ:

			i = Iron Ingot (鉄インゴット)
			g = Glass Pane (板ガラス)
			c = Clock (時計)
			S = Sticky Piston (粘着ピストン)

			ii
			gii
			cS

	EN: AAC Round
	JP: 対空通常弾

		Hand-Held Naval Gun専用の弾薬アイテム。発射ごとに1個消費する。

		・レシピA（16個分）:

			c = Clock (時計)
			I = Iron Block (鉄ブロック)
			T = TNT

			 c
			ITI
			ITI

		・レシピB（レシピAのものと同等）:

			S = Steel Block (鋼鉄ブロック)
			B = Brass Block (真鍮ブロック)

			 c
			STS
			BTB


・使用方法

	Hand-Held Naval Gun はメインハンドあるいは両手に持った状態で使用することができます（オフハンドのみの場合は不可）。
	弾薬が未装填の Hand-Held Naval Gun はインベントリに AAC Round がある状態で右クリックすると装填できます。
	弾薬を装填した状態の Hand-Held Naval Gun は右クリックすると発射できます。
	発射時インベントリに AAC Round がある場合、自動的に再装填されます。
	発射後、プレイヤーの足元に打殻の鉄インゴット（または真鍮インゴット）が1個ドロップします（サバイバルモードのみ）。
	発射後の後座・復座には1秒、弾薬の装填には3秒かかります（つまり最短で4秒/発です）。

	発射された砲弾は、ブロック（水や溶岩を含む）やmobに当たるか一定時間（最長4秒）経過すると爆発します。
	爆発はバニラのTNTブロックと範囲・威力ともに同程度ですが、ブロックは破壊しません（ただしTNTブロックは誘爆させます）。
	安全のため、発射地点から約8m以内で当たった場合は爆発せず、当たったのがmobの場合は直接ダメージのみを与えます。
	砲弾が当たった場所では、砲弾の残骸である鉄インゴットが1個ドロップします（サバイバルモードのみ）。

	Hand-Held Naval Gun を持った状態で一定距離離れたmobやブロックを左クリックすると、ロックオンすることができます。
	ロックオンされたmobや座標には緑色の□マーカーが表示されます。
	ロックオンの解除は、空を右クリックするかスニーク状態で右クリックすることでできます。
	対象を狙うときの目安位置が緑色の◇マーカーとして表示されます。
	十字カーソルを◇マーカーに合わせて撃つと当たりやすいかもしれません。
	ロックオンしている間は、◇マーカーの距離に合わせて砲弾の時限信管が自動調定されます（0.15～4秒）。
	ただし、対mobの◇マーカーは対象が現時点の方向・速度で移動を続けたらという仮定で表示しているため、外れることもあります。
	また、クライアント側の状態のみで未来位置を予測しているため、通信ラグの発生するマルチプレイ環境ではまったく役に立ちません。
	あくまで目安としてご利用ください。
