

	HandheldNavalGun 0.1.0


・このmodについて

	携帯型の火砲 Hand-Held Naval Gun と、その弾薬 AAC Round を追加します。


・前提環境（必須）

	Java 8
	Minecraft 1.10.2
	Minecraft Forge 1.10.2-12.18.3.2185-


・インストール方法

	JARファイルをmodsフォルダ内に置く


・追加されるアイテム

	EN: Hand-Held Naval Gun
	JP: 単装高角砲（携帯型）

		このmodのメインである武器アイテム。
		仰角を90度までとることができ、半自動装填装置を装備している。
		砲員のような小人が砲を動かしていると噂されるが、それはクラフター特有の幻覚症状である。

		LMB: Trace a target.
		RMB: Shoot and Reload.

		・レシピ:

			i = Iron Ingot (鉄インゴット)
			g = Glass Pane (板ガラス)
			c = Clock (時計)
			S = Sticky Piston (粘着ピストン)

			ii
			gii
			cS

	EN: AAC Round
	JP: 対空通常弾

		Hand-Held Naval Gun 専用の弾薬アイテム。装填するごとに1個消費する。
		鋼鉄製の弾殻を持つ砲弾を真鍮製の薬莢にはめ込んで一体化させたもの。
		砲弾の弾頭部分には時限信管が、弾底部分には着発信管とトレーサーが装着されている。

・使用方法

	・Hand-Held Naval Gun について
	Hand-Held Naval Gun はメインハンドあるいは両手に持った状態で使用することができます（オフハンドのみの場合は不可）。
	弾薬が未装填の Hand-Held Naval Gun は、インベントリに AAC Round がある状態で右クリックすると弾薬を装填できます。
	弾薬を装填した Hand-Held Naval Gun は、右クリックすると砲弾を発射できます。
	発射時、インベントリに AAC Round がある場合は自動的に再装填されます。
	発射後、プレイヤーの足元には打殻の鉄インゴット（または真鍮インゴット）が1個ドロップします（サバイバルモードのみ）。
	発射後の後座・復座には1秒、弾薬の装填には3秒かかります（つまり最短で4秒/発です）。

	・砲弾について
	発射された砲弾は、ブロック（水や溶岩を含む）やmobに当たるか一定時間（最長4秒）経過すると爆発します。
	爆発はバニラのTNTブロックと範囲・威力ともに同程度ですが、ブロックは破壊しません（ただしTNTブロックは誘爆させます）。
	安全のため、発射地点から約8m以内で当たった場合は爆発せず、当たったのがmobの場合は直接ダメージのみを与えます。
	砲弾が当たった場所では、砲弾の残骸である鉄インゴットが1個ドロップします（サバイバルモードのみ）。

	・射撃指揮機能について
	射撃指揮機能は指定した標的を自動で観測し、射撃する方向の目安を表示するとともに、砲弾の時限信管を自動設定する機能です。
	Hand-Held Naval Gun をメインハンドに持った状態で一定距離離れたmobやブロックを左クリックすると、標的として設定できます。
	標的となったmobやブロックの座標には緑色の□マーカーが表示されます。
	標的を解除するには、空を右クリックするか、スニーク状態で右クリックしてください。
	標的を狙うときの目安位置が緑色の◇マーカーとして表示されます。
	画面中央の十字カーソルを◇マーカーに合わせて撃つと当たりやすいかもしれません。
	標的を観測している間は、◇マーカーの距離に合わせて砲弾の時限信管が自動設定されます（0.15～4秒）。
	ただし、対mobの◇マーカーは標的が現時点の方向・速度のまま移動を続けたらという仮定で表示しているため、外れることもあります。
	また、クライアント側の状態のみで未来位置を予測しているため、通信ラグの発生するマルチプレイ環境ではまったく役に立ちません。
	あくまで目安としてご利用ください。
